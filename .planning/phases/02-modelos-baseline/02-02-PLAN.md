    ```python
    @dataclass(frozen=True)
    class OrderedLogit:
        c1: float   # cutpoint inferior (loss|draw)
        c2: float   # cutpoint superior (draw|win); invariante c1 < c2
        scale: float  # > 0
    ```
    Parametrización (documentar en docstring): con dr = diferencia de Elo desde la perspectiva del equipo A
    (incluyendo +100 si A es local no neutral) y sigma la logística:
    - P(home_win) = sigma((dr - c2) / scale)
    - P(home_win o draw) = sigma((dr - c1) / scale)
    - P(draw) = sigma((dr - c1)/scale) - sigma((dr - c2)/scale)   [>= 0 porque c1 < c2]
    - P(away_win) = 1 - sigma((dr - c1)/scale)

    `def uniform_wdl() -> np.ndarray`: return np.full(3, 1.0 / 3.0).

    `def solo_elo_probs(dr: np.ndarray, model: OrderedLogit) -> np.ndarray`: vectorizado, devuelve (n, 3)
    en orden [home_win, draw, away_win]; clip inferior 1e-12 antes de normalizar para estabilidad de log-loss.

    `def fit_solo_elo(dr: np.ndarray, outcome_idx: np.ndarray) -> OrderedLogit`: MLE con
