    2. `def neg_log_lik(params, x, y, home_idx, away_idx, is_home, w, n_teams)` — copiar VERBATIM de RESEARCH:
       layout de params = [att (n), dfn (n), c, gamma, rho];
       log_lam = c + att[home_idx] - dfn[away_idx] + gamma * is_home   (is_home = (~neutral) como float);
       log_mu  = c + att[away_idx] - dfn[home_idx];
       ll = w * (tau_log(...) + x*log_lam - lam + y*log_mu - mu); devolver -ll.sum()
       MÁS la penalización de identifiabilidad: `+ 1000.0 * (att.sum()**2 + dfn.sum()**2)`
       (la NLL tiene dirección plana sin esto — pitfall 7c; documentar en docstring).

    3. `def grad_neg_log_lik(params, ...)` — gradiente analítico (D-14 lo exige derivado; sin él cada fit toma
       ~676 evaluaciones por iteración — pitfall 7a). Derivadas concretas por partido con peso w:
       - Sea glam = ∂ll/∂log_lam = w * (x - lam + dtau_dloglam), con dtau_dloglam por caso:
         (0,0): -lam*mu*rho / (1 - lam*mu*rho); (0,1): lam*rho / (1 + lam*rho); resto: 0.
       - Sea gmu = ∂ll/∂log_mu = w * (y - mu + dtau_dlogmu), con dtau_dlogmu por caso:
         (0,0): -lam*mu*rho / (1 - lam*mu*rho); (1,0): mu*rho / (1 + mu*rho); resto: 0.
       - ∂ll/∂rho por caso: (0,0): w * (-lam*mu)/(1 - lam*mu*rho); (0,1): w * lam/(1 + lam*rho);
         (1,0): w * mu/(1 + mu*rho); (1,1): w * (-1)/(1 - rho); resto: 0.
       - Acumulación (negar todo al final porque es NEG log-lik):
         grad_att[i] = Σ glam donde home_idx==i + Σ gmu donde away_idx==i   (usar np.bincount con weights)
         grad_dfn[i] = Σ (-glam) donde away_idx==i + Σ (-gmu) donde home_idx==i
         grad_c = Σ glam + Σ gmu;  grad_gamma = Σ glam * is_home;  grad_rho = Σ ∂ll/∂rho
       - Penalización: grad_att += 2000.0 * att.sum(); grad_dfn += 2000.0 * dfn.sum().

    4. `def fit_dixon_coles(matches: pd.DataFrame, cutoff: pd.Timestamp, xi: float) -> DixonColesModel`:
       - train = matches[matches["date"] < cutoff] (corte ESTRICTO — fit-at-cutoff, Pattern 1 del RESEARCH);
       - w = np.exp(-xi * (cutoff - train["date"]).dt.days.to_numpy()); descartar filas con w < 1e-4
         (truncamiento numéricamente neutro — documentar que es equivalente al "todo el histórico" de D-07);
       - los 677 partidos con ET/shootout SE INCLUYEN en el fit de goles (decisión del Director sobre OQ5:
         incluir y documentar; efecto ~1.4% de filas);
       - índice de equipos = equipos presentes en train (orden alfabético, determinista);
       - x0: att/dfn = 0, c = log(media de goles de train), gamma = 0.1, rho = 0.0;
       - `scipy.optimize.minimize(neg_log_lik, x0, args=..., jac=grad_neg_log_lik, method="L-BFGS-B",
          bounds=[(None, None)] * (2*n_teams + 2) + [(-0.2, 0.2)])` — el bound SOLO en rho (pitfall 8:
          fuera de rango tau produce probabilidades negativas);
       - ValueError con result.message si result.success es False;
       - devolver DixonColesModel (definido en Task 2; para esta task puede definirse el dataclass mínimo ya).
